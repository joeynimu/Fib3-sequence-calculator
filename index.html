Fibonacci Sequence nth number finder app

The Challenge

Calculate the 4000th fib3 number.

Fib3 is defined as a Fibonacci sequence that uses the last 3 numbers to generate the next one (as opposed to the regular Fib)

So the first 6 numbers from this sequence will be 1 1 1 3 5 9

My approach

To solve the above problem; I generated the below variables and formular;

  • leting the nth number(4000) be a variable n;
  • we store all our numbers in an array, arr;
  • Our first three numbers will be equal to 1 hence arr[0], arr[1], arr[2] = 1 ;
  • Since the index position in arrays starts from 0 our nth number will be sitting in the position (n - 1). This position is held by a variable nIndex = n-1
  • I then do a loop and each time i calculate the next number provided our nth number is not reached yet. I then push this next number into our array referenced by thearr variable. The formular for this would be;
    nextnumber = (lastnumber - 1) + (lastnumber - 2) + (lastnumber - 3)
  • I then return the nth number from the array using arr[nIndex]

My working Code

				
                
function fiboThree(n) { 
var arr = []; 
arr[0] = 1;
arr[1] = 1;
arr[2] = 1;
var nIndex = n - 1; 
for (var c = 3; c < n; ++c) { 
arr.push( (c < 3) ? c : arr[c-1] + arr[c-2] + arr[c-3]); 
}
return arr[nIndex]; 
}
console.log( fiboThree( 4000 ) );

The Code above returns infinity

Hence the 4000th number using the fib3 formular is infinity
